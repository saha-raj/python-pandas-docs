DataFrames store tabular data in Python, commonly using pandas.

```python
import pandas as pd
df = pd.DataFrame({'Name': ['Alice', 'Bob'], 'Age': [25, 30]})
print(df)
```
```output
Name  Age
0  Alice   25
1  Bob   30
```
